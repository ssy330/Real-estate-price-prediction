# 260629 연구 논의 정리

## 오늘 확정한 내용

- baseline 논문을 확정했다.
- baseline 논문은 강남구 아파트 실거래 데이터를 기반으로 OLS 기반 헤도닉 가격모형과 Random Forest 모델을 비교한 연구이다.
- 본 연구는 baseline 논문의 최종 선택 변수 16개를 출발점으로 삼는다.

## 연구 방향

본 연구의 기본 방향은 baseline 논문에서 최종 선택된 16개 변수에 GIS 변수와 LLM 기반 변수를 추가하고, 추가 변수들이 예측 성능에 유의미한 영향을 주는지 확인하는 것이다.

즉, baseline 모델을 먼저 재현한 뒤 다음과 같은 비교를 수행한다.

1. baseline 최종 변수 16개만 사용한 모델
2. baseline 변수 16개 + GIS 변수
3. baseline 변수 16개 + LLM 변수
4. baseline 변수 16개 + GIS 변수 + LLM 변수

이를 통해 GIS 변수와 LLM 변수가 기존 변수만으로 설명되지 않는 지역적·맥락적 정보를 추가로 제공하는지 검증한다.

## GIS 변수에 대한 문제의식

기존 선행 연구에서도 GIS 관련 변수들은 대부분 후보 변수로 포함되어 있었다. 예를 들어 지하철역, 학교, 병원, 공원, 대학교, 박물관 등 주요 시설까지의 거리 변수가 후보 변수로 사용되었다.

그러나 baseline 논문에서는 이러한 GIS 변수들이 최종 변수로 거의 선택되지 못했다. 그 이유는 연구 대상 지역이 강남구로 한정되어 있었기 때문으로 보인다. 강남구 내부에서는 지역 간 GIS 환경 차이가 상대적으로 크지 않아, GIS 변수가 가격 예측 성능에 충분한 차별성을 제공하지 못했을 가능성이 있다.

## 지역 확장 필요성

우리는 GIS 변수와 LLM 기반 지역 맥락 변수가 지역별로 더 뚜렷한 차이를 만들 수 있기를 기대한다. 이를 위해 분석 대상을 강남구에만 한정하지 않고, 더 넓은 지역으로 확장하는 방향을 고려한다.

지역을 확장하면 다음과 같은 효과를 기대할 수 있다.

- 지하철 접근성, 생활 인프라, 학군, 상권, 개발 호재 등 GIS 기반 변수의 분산이 커질 수 있다.
- LLM이 포착하는 지역 이미지, 선호도, 개발 가능성, 생활권 특성 등의 차이가 더 뚜렷해질 수 있다.
- 기존 baseline 변수만으로 설명하기 어려운 지역 맥락을 추가 변수들이 보완할 가능성이 커진다.

## 지역 확장 방식

지역 확장 방식은 아직 확정하지 않았다. 현재 고려 중인 방향은 다음과 같다.

- 서울 전체에서 표본을 샘플링하는 방식
- 특정 자치구들을 선택해 비교하는 방식
- 강남구와 성격이 다른 자치구를 함께 포함하는 방식
- 가격대, 생활권, 개발 양상 등이 다른 지역군을 구성하는 방식

다만 최종 확장 전략은 baseline 논문 재현을 먼저 수행한 뒤 결정한다. 재현 모델을 실제로 돌려보면서 실행 시간, 데이터 수집 난이도, 변수 구축 가능성, 모델 성능, 샘플 수 등을 확인한 후 현실적인 범위를 정한다.

## 우선순위

현재 단계에서의 우선순위는 다음과 같다.

1. baseline 논문을 최대한 그대로 재현한다.
2. baseline의 최종 변수 16개를 동일하게 구성한다.
3. Random Forest 모델 성능이 논문과 유사하게 나오는지 확인한다.
4. 모델 실행 시간과 데이터 처리 부담을 확인한다.
5. 이후 지역 확장 범위와 GIS/LLM 변수 추가 전략을 결정한다.

## 연구 목적

본 연구의 목적은 GIS 변수와 LLM 기반 변수가 부동산 가격 예측 또는 투자 유망 지역 분석에서 유의미한 성능 개선을 가져오는지 검증하는 것이다.

특히 강남구처럼 비교적 균질한 지역에서는 GIS 변수의 영향이 제한적일 수 있으므로, 지역을 확장해 GIS 및 LLM 변수의 지역별 차이가 더 분명하게 드러나는 조건을 만들고자 한다.

최종적으로는 다음 질문에 답하는 것을 목표로 한다.

- baseline 변수만 사용했을 때와 비교해 GIS 변수를 추가하면 성능이 개선되는가?
- baseline 변수만 사용했을 때와 비교해 LLM 기반 변수를 추가하면 성능이 개선되는가?
- GIS 변수와 LLM 변수는 서로 다른 정보를 제공하는가?
- 지역 범위를 확장했을 때 두 변수군의 효과가 더 뚜렷해지는가?
